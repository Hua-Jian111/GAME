# 🐍 贪吃蛇

一个纯 HTML/CSS/JS 实现的经典贪吃蛇游戏，单文件即可运行。

## 玩法

- **方向键** 或 **WASD** 控制蛇的移动方向
- **空格键** 暂停 / 继续
- 吃到红色食物 +10 分，蛇身变长
- 撞墙或撞到自己游戏结束

## 特性

- 三档速度切换（慢速 / 普通 / 快速）
- 最高分本地持久化（localStorage）
- 蛇头带方向眼珠动画
- 蛇身渐变透明效果
- 游戏结束与暂停遮罩提示

## 快速开始

直接用浏览器打开 index.html 即可。

`ash
start index.html
`

或在本地启动一个静态服务：

`ash
npx serve .
`
